# Pedro Pessanha — Data Science Portfolio

Olá! Sou o Pedro Pessanha, estudante de Ciência de Dados pela FIAP 
com formação prevista para 2026. Esse repositório reúne os projetos 
práticos que desenvolvi ao longo do curso, cobrindo desde análise 
exploratória e machine learning até modelagem dimensional e visualização 
de dados.

Atualmente atuo como Analista de Suporte N1 na Minsait, atendendo 
o cliente Petrobras, onde desenvolvi visão analítica monitorando KPIs 
como CSAT e IBL. Estou em transição para a área de dados e esses 
projetos refletem essa jornada.
