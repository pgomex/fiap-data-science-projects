# 📊 Data Viz & Data Science — Melhores Compras

Projeto desenvolvido durante o Tecnólogo em Ciência de Dados na FIAP, 
com foco em visualização de dados e ciência de dados aplicada a um 
e-commerce fictício chamado Melhores Compras.

---

## 🧠 Contexto

A Melhores Compras precisava entender o comportamento de vendas da linha 
PetShop ao longo de 5 anos e otimizar seus investimentos em publicidade. 
Dois desafios distintos, duas soluções diferentes.

---

## 📁 O que tem aqui

### 1. Dashboard Estratégico de Vendas (Power BI)
Análise de 250 mil registros de vendas entre 2020 e 2024, com foco em:
- Tendências de vendas por região, produto e período
- Formas de pagamento mais utilizadas
- Performance por centro de distribuição
- KPIs estratégicos para a área comercial

### 2. Regressão Linear para ROI de Marketing
Modelo preditivo desenvolvido em Python para entender o impacto de cada 
canal de publicidade nas vendas. O objetivo era responder uma pergunta 
simples mas valiosa: onde vale mais a pena investir?

Técnicas utilizadas:
- Análise de correlação entre variáveis
- Variáveis dummy para categorias de mídia
- Regressão Linear com avaliação por R²
- Visualizações com Matplotlib e Seaborn

---

## 🛠️ Tecnologias

- Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn)
- Power BI
- Excel

---

## 📌 Principais resultados

- Identificação dos canais de mídia com melhor retorno sobre investimento
- Dashboard interativo com visão estratégica de 5 anos de vendas
- Modelo de regressão capaz de prever variações de vendas com base 
nos investimentos em publicidade
